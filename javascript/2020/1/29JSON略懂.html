JSON（JavaScript Object Notation）
            是轻量级的数据交换格式
            独立于语言（JSON 的语法是来自 JavaScript 对象符号的语法，但 JSON 格式是纯文本。读取和生成 JSON 数据的代码可以在任何编程语言编写的。）
            是“自描述的”且易于理解。
            JSON是储存和传输数据的格式，经常在数据从服务器发送到网页是使用。
        JSON书写格式：
            名称/值对，由双引号中的字段名构成，中间有冒号
            "firstName":"Name"
        JSON对象：
            花括号，能够包含多个  名称/值对
            {"firstName":"Name","lastName":"Name"}
        JSON数组:
            方括号，能够包含对象
            "employees":[
                {"firstName":"Bill", "lastName":"Gates"}, 
                {"firstName":"Steve", "lastName":"Jobs"}, 
                {"firstName":"Alan", "lastName":"Turing"}
            ]
